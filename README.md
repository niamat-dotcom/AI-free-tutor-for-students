# AI-free-tutor-for-students
I built an offline-first AI-inspired tutoring system  to support exam preparation for students with limited internet access. The project focuses on accessibility, educational impact, and intelligent system design
An **offline-first, exam-focused intelligent tutoring system** built with Streamlit to support students preparing for:
- SAT (English & Math)
- Duolingo English Test (DET)
- Python Programming

## Motivation
Many students in low-resource regions face unreliable internet access.  
This project provides a **free, offline-capable AI-inspired tutor** to ensure continuous learning.

##  Features
- Offline knowledge-based tutoring
- Exam-oriented explanations
- Topic analytics
- PDF export for lessons
- Clean and simple UI

## Tech Stack
- Python
- Streamlit
- FPDF

## Target Users
- High school students
- Exam candidates (SAT, DET)
- Beginner programmers

##  Future Improvements
- ML-based answer generation
- Adaptive difficulty
- Real-time feedback
- Cloud + offline hybrid mode
